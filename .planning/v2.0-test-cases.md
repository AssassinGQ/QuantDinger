## 3. API 对比

| 功能 | 外部库 (v1.0) | 内部Client (v2.0) | 变更 |
|------|--------------|------------------|-----|
| 合约创建 | `make_contract(SymbolConfig)` | `_create_contract(symbol, market_type)` | ✅ 新方法 |
| 认证 | `qualify_contract(contract)` | `_qualify_contract_async(contract)` | ✅ async |
| 历史K线 | `get_historical_bars(contract, timeframe)` | `get_historical_bars(symbol, timeframe, limit)` | **需新增** |
| 实时报价 | `get_ticker_price(contract)` | `get_quote(symbol, market_type)` | ✅ 已存在 |
| 连接管理 | 独立实例 | 单例 `get_ibkr_client()` | ✅ 复用单例 |
