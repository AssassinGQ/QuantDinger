## Product text (must appear in code comment near PartiallyFilled handler)

`PartiallyFilled → 累计值覆盖 DB 的 filled/remaining（不做增量计算）`

## Invariants (implement)

- Let `total_q = float(trade.order.totalQuantity or 0)`, `f = float(trade.orderStatus.filled or 0)`, `r = float(trade.orderStatus.remaining or 0)`.
- Log **warning** if `f` &lt; `ctx.last_reported_filled` (non-decreasing expectation).
- Log **warning** if `abs(f + r - total_q) > max(1e-4, 1e-6 * max(1.0, total_q))` (epsilon guard; document constant as `_ORDER_QTY_EPS` in client.py).
- After handling, set `ctx.last_reported_filled = f` on the context object (add field `last_reported_filled: float = 0.0` to `IBKROrderContext`).
